# DocConverter

A powerful document format converter built with Node.js. Supports conversion between Markdown, HTML, PDF, DOCX, and more.

## Features

- Convert between multiple document formats
- Batch conversion support
- Preserves formatting and styles
- CLI and programmatic API
- DevOps-ready with Docker, CI/CD

## Installation

```bash
npm install
npm link
```

## Usage

```bash
doc-convert input.md output.html
doc-convert --format pdf input.md
```
