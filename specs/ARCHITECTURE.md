## Components

### 1. CLI Interface
- Command-line argument parsing
- File path handling
- Batch processing queue

### 2. Converter Pipeline
- Input format detection
- Conversion chain execution
- Output format routing

### 3. Format Engines
- **markdown-it**: Markdown вҶ’ HTML
- **pdfkit**: HTML вҶ’ PDF
- **docx**: HTML вҶ’ DOCX
- **puppeteer**: HTML вҶ’ PDF (alternative)

### 4. Format Preserver
- Style extraction and apply
- Image handling
- Table conversion
